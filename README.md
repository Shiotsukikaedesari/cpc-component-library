# cpc-component-library

> 作为CPC官网的组件库支撑

## Build Setup

### 项目启动

`npm install`

`npm run dev`

### 热部署地址： 

`localhost:8080`

### 项目打包

`npm run build`

### 查看打包的报告

`npm run build --report`

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report
```
